iPA Generation Process
Topics Covered :
  • Microphone Access (For audio video and recorder)
  • Location Access Permission
  • Calendar Access Permission
  • Contacts Permission
  • Camera and Photo library access Permission
  • To access Custom URL’s
  • Push Notification
  • Certificates & Provisioning Profile Generation
For Microphone :-
  • Privacy - Microphone Usage Description :: This app requires microphone access to function properly.
For Motion & Location :-
  • Privacy - Motion Usage Description :: This app requires motion detection access to function properly.
  • NSLocationAlwaysUsageDescription :: This app requires constant access to your location in order to track your position, even when the screen is off.
For Calendar :-
  • Privacy - Calendars Usage Description :: This app requires calendar access to function properly.
For Contacts :-
  • Privacy - Contacts Usage Description :: $(PRODUCT_NAME) contact use. OR This app requires calendar access to function properly.
For Microphone :-
  • Privacy - Microphone Usage Description :: This app requires microphone access to function properly.
For Camera and Photo library :-
  • Privacy - Camera Usage Description :: $(PRODUCT_NAME) Camera Use
  • Privacy - Photo Library Usage Description :: This app requires photo library access to function properly.

Push Notification, Certificate & Provisioning Profile Generation Process
Create your App Id with as the description provided over the page :
  • App Id Description
  • App Id prefix
  • App Id Suffix - Select Explicit App ID - Bundle Id (Your application Bundle Id here)
  • App Services - Select what features you like to include in your application. Ex. Push Notification: Make sure its checked if your application is using Push notification
Once done with all the details select on Continue. It will navigate to next page for the confirmation of app id Register it.
Select according to your need to create certificates and provisioning profile -
  • Developer certificate - To run application on your device.
  • Distribution certificate - To upload your application on app store
  • To share your application except from appstore (For diawi, UAT etc)
Go to ‘Certificates’ Section - All Click on + Button. A Ui will be poped up like below -
Open the cerficates and provisioning profile from download list, it will be listed in keychain and Xcode automatically.
