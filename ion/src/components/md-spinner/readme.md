Android Inspired Material Design Spinner
=======

## Templet Syntax:
```
<md-spinner [state] = "spinner"></md-spinner>
<md-spinner [state]="spiner" [type]="'fixed'"></md-spinner>
```
## Inputs
```
1. state:boolean

[true/false]

2. type:string

['inline' -> Inline loader]
['fixed' -> Fixed Center Screen loader]

```
